**Tasker Auto‑Switch Setup**
Follow these instructions to make Android switch to HP H150 automatically.

1. Install Tasker
From Google Play Store.

2. Create Profile: Auto‑connect on call
Profile: Call → Incoming OR Outgoing
Action: Bluetooth → Connect → HP H150

3. Profile: Auto‑connect on media
Profile:
Event → Media → Audio Becoming Noisy or
Applications → (YouTube, Spotify, Netflix)
Action: Bluetooth → Connect → HP H150

4. Profile: Auto‑disconnect when Windows is active
Profile:
State → Bluetooth Connected → (invert → NOT HP H150)
Action: Bluetooth → Disconnect → HP H150
